โครงงานครั้งนี้มีวัตถุประสงค์ เพื่อ
1. ศึกษาวิธีการต่อวงจรบนบอร์ด Arduino
2. สร้างของเล่นที่สามารถนำมาเล่นแข่งกันเพื่อความสนุกได้
3. นำความรู้ที่ได้จากการทำโครงงานนี้ไปประยุกต์ใช้ต่อในอนาคต
โดยอุปกรณ์หลักที่พวกเราเลือกใช้คือ
1. บอร์ด Arduino Uno
2. เซนเซอร์ตรวจจับเสียง KY-037
3. หลอดไฟ LED
4. เซนเซอร์วัดระยังทาง HC-SR04
หลักการทำงานของสิ่งประดิษฐ์ของกลุ่มพวกเราคือ เมื่อผู้เล่นทำการเปล่งเสียงออกมาจะทำให้หลอดไฟสว่าง
โดยหลอดไฟจะมี 3 ระดับ และจะสว่างตามความดังของเสียงผู้เล่นจากน้อยไปมาก โดยต้องเปล่งเสียงในระยะห่างที่กำหนดเอาไว้

รายละเอียดโครงงาน

โครงงานนี้เป็นโครงงานที่ใช้เทคโนโลยีและอุปกรณ์อิเล็กทรอนิกส์เพื่อสร้างของเล่นที่สนุกและสอนผู้เล่นเรื่องการใช้บอร์ด Arduino Uno
ในการสร้างอุปกรณ์อัจฉริยะที่ตอบสนองตามเสียงที่ผู้เล่นเปล่งออกมา โดยมีวัตถุประสงค์หลัก 3 ข้อดังนี้:

1. ศึกษาวิธีการต่อวงจรบนบอร์ด Arduino Uno: โครงงานนี้เป็นโอกาสที่ผู้ร่วมทำโครงงานจะได้เรียนรู้การใช้บอร์ด Arduino Uno
ในการสร้างโปรแกรมและต่อวงจรเพื่อควบคุมอุปกรณ์อิเล็กทรอนิกส์ เช่น เซนเซอร์และ LED ในโครงงานนี้ ผู้ร่วมโครงงานจะได้ศึกษาเรื่องการโปรแกรม Arduino และการต่อวงจรอิเล็กทรอนิกส์ที่เกี่ยวข้องกับการตรวจจับเสียงและควบคุมหลอด LED ตามระดับเสียงที่ตรวจจับได้

2. สร้างของเล่นที่สามารถนำมาเล่นแข่งกันเพื่อความสนุกได้: ผู้ร่วมโครงงานจะใช้ความรู้และทักษะที่เรียนรู้จากการศึกษา
และทดลองทำบน Arduino Uno เพื่อสร้างอุปกรณ์เล่นที่สนุก โดยอุปกรณ์นี้สามารถเล่นแข่งกันได โดยการเปล่งเสียง และต้องควบคุมระดับแสงของหลอด LED ตามระดับเสียงของผู้เล่น

3. นำความรู้ที่ได้จากการทำโครงงานนี้ไปประยุกต์ใช้ต่อในอนาคต: โครงงานนี้ไม่เพียงเพื่อสร้างของเล่นสนุก
แต่ยังเป็นโอกาสที่ผู้ร่วมโครงงานจะได้รู้จักการใช้ Arduino Uno ในโปรเจคอื่น ๆ ในอนาคต นอกจากนี้ การเรียนรู้เรื่องการตรวจจับเสียงและควบคุมแสง ก็สามารถนำไปใช้ในโปรเจคอื่น ๆ ได้ เช่นการสร้างอุปกรณ์ควบคุมเครื่องใช้ในบ้านอัจฉริยะหรือโครงงานที่เกี่ยวกับเสียงและแสงในอนาคต

การทำงานของอุปกรณ์คือเมื่อผู้เล่นเปล่งเสียงออกมาผ่านเซนเซอร์ตรวจจับเสียง KY-037 อุปกรณ์จะตรวจจับระดับเสียงและควบคุมหลอดไฟ LED ให้สว่างตามระดับเสียงที่ตรวจจับได้ โดยมี 3 ระดับที่แสดงด้วยความสว่างของ LED
ผู้เล่นจะต้องเปล่งเสียงในระยะห่างที่กำหนดเอาไว้เพื่อเพิ่มความสนุกในการแข่งขันและทดสอบทักษะในการควบคุมเสียงและแสง

1. ศึกษาวิธีการต่อวงจรบนบอร์ด Arduino ในขอบขางนี้ขอเสนอการดำเนินการตามลำดับเพื่อศึกษาวิธีการต่อวงจรบนบอร์ด Arduino:
1.1 ศึกษาเกี่ยวกับ Arduino: ศึกษาเกี่ยวกับบอร์ด Arduino Uno, การต่อวงจร, และภาษาโปรแกรม Arduino IDE
1.2 ศึกษาวงจรเซนเซอร์: รู้จักเซนเซอร์ตรวจจับเสียง KY-037 และเซนเซอร์วัดระยะทาง HC-SR04 อย่างละเอียด
1.3 ศึกษาการใช้งาน LED: เรียนรู้วิธีการใช้งานและควบคุมหลอดไฟ LED บน Arduino

2. สร้างของเล่นที่สามารถนำมาเล่นแข่งกันเพื่อความสนุกได้ ของเล่นที่คุณกำลังสร้างมีรายละเอียดการทำงานดังนี้:
2.1 ระบบตรวจจับเสียง: ใช้เซนเซอร์ KY-037 เพื่อตรวจจับเสียงที่ผู้เล่นเปล่งออกมา
2.2 การประมวลผลเสียง: ใช้ Arduino Uno เพื่อประมวลผลสัญญาณเสียงและระบุระดับความดังของเสียง
2.3 ควบคุมแสง LED: ควบคุมหลอดไฟ LED ในโหมด 3 ระดับตามความดังของเสียงที่รับมา

3. นำความรู้ที่ได้จากการทำโครงงานนี้ไปประยุกต์ใช้ต่อในอนาคตเพื่อให้โครงงานมีความหมายมากขึ้นในการพัฒนาความรู้และทักษะ
คุณสามารถพิจารณาเรื่องต่อไป:
3.1 พัฒนาแอพพลิเคชันเสริม: พัฒนาแอปพลิเคชันสำหรับมือถือหรือคอมพิวเตอร์ที่อาจเชื่อมต่อกับโครงงานของคุณผ่าน
Bluetooth หรือ Wi-Fi เพื่อควบคุมการเปล่งแสง LED หรือแสดงข้อมูลเสียงออกมา
3.2 การสร้างโครงงานที่ใช้เซนเซอร์: ใช้ความรู้ที่ได้จากการใช้เซนเซอร์ KY-037 และ HC-SR04
เพื่อสร้างโครงงานอื่น ๆ ที่เกี่ยวข้องกับการตรวจจับเสียงหรือการวัดระยะ
3.3 การพัฒนาทักษะการเขียนโปรแกรม: เรียนรู้เพิ่มเติมเกี่ยวกับการเขียนโปรแกรม Arduino และการโปรแกรมแบบ embedded systems
เพื่อประยุกต์ใช้ในงานที่มีการควบคุมอุปกรณ์โดยใช้ไมโครคอนโทรลเลอร์

โดยหลักการทำงานของชิ้นงานนี้มีดังนี้ :
1. ถ้าระยะของตัวคุณห่างจากชิ้นงาน 12 เซนติเมตร หรือ ใกล้เกิน 5 เซนติเมตร จะไม่สามารถเล่นได้ แต่หากอยู่ในระยะที่กำหนด LED ขึ้นสีฟ้าและสามารถเล่นได้
(หลักการคำนวนดังนี้ cm = (pulseWidth / 2.0) / 29.1)
2.ถ้าเสียงอยู่ในช่วงที่มากกว่า 570 เดซิเบล RGB LED อันที่ 1 จะมีเขียวขึ้น ถ้าเสียงอยู่ในช่วงที่มากว่า 650 เดซิเบล RGB LED อันที่ 1 จะมีสีเขียว และ
RGB LED อันที 2 จะมีสีเหลือง ถ้าเสียงอยู่ในช่วงที่มากว่า 800 เดซิเบล RGB LED อันที่ 1 จะมีสีเขียว และ RGB LED อันที 2 จะมีสีเหลือง และ RGB LED อันที่ 3 จะมีสีแดง
